## ⑦質問・疑問・感想、シェアしたいこと等なんでも

- 授業と同様のjQueryを読み込むと発生するかもしれないエラー（自分だけの可能性もありますが…）

- 解決策：別のhttpsのjQueryを読み込ませると解決する可能性あり
